API Documentation: For Developers

If users make a GET request to the URL /api/isbn route, where isbn is an ISBN number, the website should return a JSON response containing the book's title, author, publication date, ISBN number, review count, and average score. The resulting JSON should follow the format:



{

"title": "Memory",

"author": "Doug Lloyd",

"year": 2015,

"isbn": "1632168146",

"review_count": 28,

"average_score": 5.0

}
